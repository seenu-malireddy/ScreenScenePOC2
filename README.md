# ScreenScene

A movie and TV show website made with React, TypeScript, and Firebase Authentication.

## What it does

- Browse movies and TV shows
- Search for movies
- Sign up and login with Firebase Auth
- Save favorites
- Add reviews
- Works in English and Arabic

## How to run

1. Install stuff:
```
npm install
```

2. Start the app:
```
npm run dev
```

3. Open http://localhost:3000

## What I used

- React
- TypeScript  
- Vite
- Tailwind CSS
- Firebase Authentication
- Movie database API

## Features

- Home page with movies
- Search page
- Movie detail pages
- User accounts
- Favorites list
- Reviews
- Language switching
